# Hindidub 🇮🇳

**YouTube English Videos to Hindi Dubbing**

Convert English audio into natural Hindi voice with just one click.

## ✨ Current Features
- Extract audio from YouTube videos
- English Speech-to-Text conversion (Whisper)
- Basic English to Hindi translation
- Audio & Video processing using **FFmpeg**
- Simple web interface (Python + HTML)

## 🛠️ Technologies Used
- Python
- HTML + CSS
- Flask (Web Framework)
- FFmpeg (Audio & Video Processing)
- Whisper (Speech-to-Text) — to be fully integrated soon

## 🚀 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/souravjana579/hindidub.git

# Install dependencies

# Run the application
